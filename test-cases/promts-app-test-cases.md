# Promts.exe — Test Cases

---

## TC-001 Application Launch

### Preconditions
Application is installed

### Steps
1. Open Promts.exe

### Expected Result
Application launches successfully and main window is displayed

---

## TC-002 Upload Valid TXT File

### Preconditions
Application is running

### Steps
1. Click "Обрати файл"
2. Select valid TXT file
3. Click "Open"

### Expected Result
TXT file is uploaded and text is displayed in input area

---

## TC-003 Run Text Processing

### Preconditions
TXT file uploaded

### Steps
1. Upload TXT file
2. Click "Запустити"

### Expected Result
Processed text appears in result area

---

## TC-004 Clear Input Text

### Preconditions
Input text exists

### Steps
1. Click "Очистити"

### Expected Result
Input area becomes empty

---

## TC-005 Export Result

### Preconditions
Processed result exists

### Steps
1. Click "Експорт"

### Expected Result
Result file is exported successfully
