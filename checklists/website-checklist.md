# Website QA Checklist — spirit-develop.org

## Functional Testing

- [ ] Homepage opens correctly
- [ ] All menu items work
- [ ] Internal links work
- [ ] External links work
- [ ] Contact forms submit correctly
- [ ] Validation messages appear
- [ ] Buttons are clickable
- [ ] Images load correctly
- [ ] Videos work correctly

## UI / UX

- [ ] Text is readable
- [ ] Buttons have consistent styles
- [ ] No broken layout
- [ ] Spacing looks correct
- [ ] Important sections are visible
- [ ] Navigation is understandable

## Responsive Testing

- [ ] Mobile layout works
- [ ] Tablet layout works
- [ ] Desktop layout works
- [ ] Buttons are easy to tap
- [ ] No overlapping elements

## Content Testing

- [ ] No spelling mistakes
- [ ] Headers are consistent
- [ ] Text formatting is correct
- [ ] No visual overload

## Basic Technical Testing

- [ ] Pages load fast
- [ ] No console errors
- [ ] No broken images
- [ ] No broken links
